Welcome to Mr O'Shea's website!

This is an example of the type of thing you're going to be building over the next few lessons. There is a Rubric on Google Classroom with the features you'll need to implement.

Here is a list of things to include:

  • At least 4 HTML pages that link together in some fashion
  • A CSS file setting the style of the pages (including font, colours, backgrounds, image size and layout)
  • An application of Javascript: maybe a button that changes the content, or some navigation that responds to the user
  • At least one image on a page
  • Use of Heading tags/links to external pages
